
# Cowabunga Pizza – Windows Server Infrastructure Project

## Project Overview

This project implements a centralized Windows Server infrastructure for a growing organization.  
The environment includes Active Directory Domain Services, Group Policy enforcement, DHCP configuration, and structured access controls.

The goal was to create a secure and scalable infrastructure that simplifies network administration and improves authentication and access management.

---

## Infrastructure Components

- Active Directory Domain Services (AD DS)
- Multiple Domain Controllers
- DHCP configuration
- Group Policy security enforcement
- File server with folder redirection
- Storage quotas using FSRM
- Windows Admin Center management

---

## Key Skills Demonstrated

- Windows Server administration
- Active Directory design
- PowerShell automation
- Network configuration
- Group Policy security management
- Infrastructure documentation
